# 🧰 Workshop Inventory

A simple inventory management system for a mechanical workshop. It allows managing tools, consumables, and stock movements in a structured and visual way.

---

## 🚀 Tech Stack

- Next.js (App Router)
- TypeScript
- Firebase (Authentication, Firestore, Storage)
- React Hooks
- CSS

---

## 📦 Features

- 🔐 Google Authentication (Firebase Auth)
- 📋 CRUD for tools and consumables
- 📊 Real-time stock tracking
- 🔁 Stock movements (IN / OUT)
- 🖼️ Image support for items (Firebase Storage)
- 🧠 Dynamic item specifications (JSON-based)
- 🧪 Mock data support for development
